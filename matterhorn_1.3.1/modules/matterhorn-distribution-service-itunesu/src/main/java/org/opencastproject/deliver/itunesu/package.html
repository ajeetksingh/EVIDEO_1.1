Media delivery and removal actions for iTunes U using iTunes U Web service.

The iTunes delivery service is calling the iTunes U Web service to upload the media content. The service will create an authorization token given the credentials specified in the configuration file, request an upload URL using the authorization token, and upload the media file or Web service XML document to the upload URL using a HTTP POST request. The service will also return the handle of the uploaded track under the given group, or the results of the Web service operation, depending on the type of the task.

  • Creating an iTunes U Web Services Document
for details on using iTunes U Web service to perform various media and group manipulation tasks.
